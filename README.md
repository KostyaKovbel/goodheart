# TT Good Heart Form

[Demo Link](https://KostyaKovbel.github.io/goodheart)

## Local development

### Dependencies
* Node v12.16.3 and higher
* NPM v6.14.4 and higher


### Installing
* Fork and clone this repository
* Run `npm install` in your terminal
* Run `npm start`

#### Tech-stack
* React.js
* TypeScript
* HTML
* Sass
* BEM methodology
